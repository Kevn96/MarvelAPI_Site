
Seja Bem Vindo ao Projeto API Marvel!

--OBJETIVO--
O intuito de desenvolver o site não é só para a pessoa ver seus personagens
favoritos, mas também ver HQs relacionados ao mesmo.
No final da pagina, após a pesquisa havera diversos links para o usuario ler
o conteudo solicitado.

--DEPENDÊNCIAS DO PROJETO!!--
O projeto a principio foi desenvolvido em:
-Python(backend)
-HTML, CSS e Bootstrap(front)

Externos usado no projeto:
- Flask (em especifico essas extensões Flask, render_template, request)
- Marvel (a API que fez tudo girar)
- BeautifulSoup (buscar informações em um site de comics para o user ler)

--INSTALAÇÃO!--
Caso queira editar, peço que siga esses passos para instalar as dependências:
-pip install flask
-pip install marvel
-pip install BeautifulSoup

Versão Utilizada:
Python 3.9.13
Flask 2.3.1
